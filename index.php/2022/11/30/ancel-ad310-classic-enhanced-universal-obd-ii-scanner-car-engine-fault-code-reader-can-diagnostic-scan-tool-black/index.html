Where's the OBDII interface?

The OBD2 port is often under the dash, or under a panel below the handbrake in the central console.

How to get started?

Key operation: 1. turn the ignition on without starting the car; 2. Start the car.

Keyless operation: 1. press start button - press again (do not press brake pedal) - insert tool and read; 2. press brake pedal and start button.

Why I can't erase the fault code?

The fault causing the fault code to show will need to be repaired first then code can be erased.

Will it diagnose ABS or SRS?

Unfortunately, no. The main function of AD310 scanner is to detect the engine system, read and clear the fault codes, but Ancel AD610 can do ABS and SRS.

What should I pay attention to before buying?

Two things you should know: 1. it works only on cars with 12V battery; 2. your car must be standard OBD II protocol (16pin DLC).
